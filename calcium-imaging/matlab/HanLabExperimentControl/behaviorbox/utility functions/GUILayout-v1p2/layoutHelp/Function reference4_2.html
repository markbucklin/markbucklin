4.2: layoutVersion  

Get the toolbox version number and date

V = layoutVersion( )
returns the version number as a string.
[V, D] = layoutVersion( )
returns both the version string and the date of creation (format is ISO8601, i.e. YYYY-MM-DD)

For example:

[v,d] = layoutVersion()

v = '1.0'
d = '2010-05-28'
